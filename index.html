Direção de Produto

Concepção, arquitetura e direção · execução pelo squad

SGT — Sistema Global Tech (HUB de Sistemas)

Direção IA Aplicada

Sistema central da Personal Global: hub que unifica gestão educacional, comercial, financeira e operacional em uma única plataforma web. Construído em Next.js 15, TypeScript e Supabase, com 17 sub-módulos em produção — incluindo check-in via QR Code, copiloto de IA para WhatsApp, analytics, dashboard financeiro, gestão de eventos e módulo educacional. Sob minha direção arquitetural, com squad executando sob backlog quinzenal.

Impacto: Centralizou em uma interface o que antes vivia em planilhas e sistemas isolados, eliminando retrabalho e estabelecendo fonte única da verdade entre as áreas.
Stack: Next.js 15 · TypeScript · Supabase · Tailwind · Vercel Papel: Direção técnica + arquitetura Ver deploy →

Atendimento Inteligente via WhatsApp (UZApi)

Direção IA Aplicada

Plataforma de atendimento construída em React, TypeScript e Supabase que integra um agente de IA para triagem, classificação e resposta automatizada de mensagens via WhatsApp. Arquitetura event-driven com webhooks. Concebido por mim para escalar atendimento sem sacrificar a qualidade humana das interações que de fato exigem julgamento.

Impacto: Automação da triagem inicial libera o time humano para casos que realmente exigem empatia ou decisão complexa.
Stack: React · TypeScript · Supabase · UZApi · OpenAI Padrão: Event-driven · webhooks · escalonamento inteligente GitHub →

Plataformas Educacionais (Portal Aluno · Portal Uploaders · Educacional)

Direção EdTech

Conjunto de produtos digitais para o ecossistema educacional da Personal Global: Portal do Aluno, Portal de Uploaders (gerenciamento de matrículas, gravações e métricas) e plataforma Educacional. Direção de produto, definição de roadmap e governança técnica sob minha responsabilidade — execução conduzida pelo squad.

Impacto: Verticais educacionais ganharam plataforma própria, com métricas de educacional, ROAS e fila de produção visíveis em tempo real.
Stack: Next.js · TypeScript · Supabase · Vercel Papel: Direção + governança técnica

Construção Primária

Código direto · decisão técnica · entrega em produção

Plataforma Interna Global

Construção Produção

Plataforma interna que organiza os processos entre comercial, guia do aluno e financeiro. Construída do zero, com backend em JavaScript/Node.js, gestão de fluxos cross-funcional e auditoria de acessos. Principal frente de construção pessoal — código próprio em todas as camadas.

Impacto: Tornou auditável e cross-funcional um processo que antes dependia de mensagens dispersas e planilhas paralelas. Pipeline visual e histórico por aluno.
Stack: JavaScript · Node.js · Express · Supabase · Service Accounts Contribuição: 100 commits diretos

Meeting Mastermind — Plataforma de Agentes de IA

Construção IA Aplicada

Plataforma flexível para criação e orquestração de agentes de IA voltados à gestão de equipes e à otimização de projetos e processos. Cada agente é configurado para um ritual ou frente específica — desde captura e estruturação de reuniões até follow-up automatizado de compromissos — com integrações nativas a ClickUp, Google Drive, Google Calendar, WhatsApp e transcrição de gravações. Concepção e execução próprias, do desenho do produto à entrega em produção.

Impacto: Transforma reuniões e interações dispersas em ações estruturadas, atribuídas e rastreáveis dentro do ClickUp — fechando o loop entre decisão e execução, e devolvendo tempo de gestão à liderança.
Stack: React · TypeScript · OpenAI · ClickUp API · Google APIs · WhatsApp API · transcrição Diferencial: Agentes configuráveis · multi-integração · multi-canal

Sistema de Renovação UPL Pro (pos08-rocket)

Construção Financeiro

Plataforma interna com painel de CS (tabela de alunos, filtros avançados, ações rápidas) e Kanban visual de 6 colunas com drag-and-drop, timer de urgência por cor e rastreamento automático de etapas. Backend Node.js/Express, Google Sheets como banco via Service Account, auditoria completa de cada ação registrada na planilha.

Impacto: Substituiu o processo manual de renovação por pipeline visual com visibilidade total e histórico auditável de cada interação.
Stack: React · TypeScript · Node.js/Express · Google Sheets API · dotenv-vault Contribuição: 53 commits diretos

Dashboards Estratégicos (Lançamento + Financeiro)

Construção Analytics

Dois painéis analíticos construídos para diferentes frentes: dashboard de lançamento (acompanhamento de métricas comerciais durante a janela de captação) e dashboard financeiro (MRR, indicadores e visualização de receita). Fontes integradas via Google Sheets, Conta Azul e ASAAS, com decisões executivas baseadas nos dados expostos.

Impacto: Tomada de decisão durante lançamentos e fechamento financeiro deixou de depender de relatórios estáticos e ganhou visualização em tempo real.
Stack: TypeScript · Next.js · Google Sheets · Conta Azul · ASAAS Contribuição: 65 commits diretos

Check-in Revolution Event

Construção Eventos

Sistema de controle de entrada para evento presencial de grande porte com leitura de QR Code, validação em tempo real contra base de inscritos e relatórios de fluxo. Construído em HTML/JavaScript com integração direta ao backend de eventos.

Impacto: Substituiu o processo manual de credenciamento por leitor QR funcional, cortando filas e reduzindo erros de identificação.
Stack: HTML · JavaScript · QR Code · Google Sheets API Contribuição: 23 commits diretos

Formulário de Contratos (Personal Global)

Construção Contratos

Interface web para preenchimento estruturado de dados destinados à geração automatizada de contratos digitais. Validação no formulário antes de qualquer geração; integrado ao pipeline n8n que cria, envia e monitora contratos via Autentique.

Impacto: Eliminou o retrabalho de cancelamento e reenvio de contratos com dados incorretos, recuperando aproximadamente 2 a 3 horas semanais do time de contratos.
Stack: HTML · JavaScript · integração Autentique + n8n Contribuição: 15 commits diretos GitHub →

Formulários Estratégicos · Revolution & UPL

MVP Marketing

Cada um construído sob demanda com prazo curto e entrega direta para a frente que solicitou.

Impacto: Resposta rápida (dias, não semanas) a demandas táticas das áreas de marketing, comercial e eventos sem onerar o backlog estratégico do squad.

Plataforma de Automações n8n

60+ workflows distribuídos em duas instâncias n8n auto-hospedadas, organizados por domínio de negócio. Tratada como produto interno com versionamento, observabilidade e governança. Abaixo as principais frentes em produção.

⚡
Sistema Revolution 2025 (13+ workflows)
Pipeline completo para evento presencial: venda via Guru/Eduzz, geração de e-tickets personalizados, controle de entrada via QR Code, gamificação com indicações, premiação automatizada e comunicação em massa via WhatsApp. Cada etapa é um workflow independente com tratamento de erros e logs.
Marketing · CS · Operação de Eventos

Pipeline de Contratos com IA (Autentique)
Ciclo completo de contratos: formulário web → criação no Autentique → envio para assinatura → webhook de status → leitura do contrato assinado com IA para extração de dados estruturados → atualização automática no banco. Usado diariamente.
